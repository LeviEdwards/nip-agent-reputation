About NIP-XX

This NIP defines a standard for publishing, querying, and verifying reputation attestations for autonomous agents operating on the Lightning Network.

⚡ Settlement-Anchored

Reputation is derived from actual Lightning payment settlements — not social proof, not self-reported claims.

📊 Raw Dimensions

No composite scores. Individual metrics (success rate, response time, uptime) let queriers weight what matters to them.

⏳ Decay by Default

Attestations lose weight over time via configurable half-life. Old reputation fades — only recent behavior matters.

🔗 Three Trust Tiers

Bilateral (1.0) — direct counterparty · Observer (0.7) — third-party monitor · Self (0.3) — self-reported
